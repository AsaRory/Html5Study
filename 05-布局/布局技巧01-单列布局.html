1 水平居中

1.1 使用 inline-block 和 text-align实现

.parent {
  text-align: center;
}
.child {
  display: inline-block;
}
优点：兼容性好；
不足：需要同时设置子元素和父元素
水平居中

1.2 使用 margin:0 auto 实现

.child {
  width: 200px;
  margin:0 auto;
}
优点：兼容性好
不足：需要指定宽度
水平居中

1.3 使用table实现

.child {
  display: table;
  margin: 0 auto;
}
优点：只需要对自身进行设置
不足：IE6,7需要调整结构
水平居中

1.4 使用 绝对定位 实现

.parent{
  position: relative;
}
.child{
  position: absolute;
  left: 50%;
  transform: translate(-50%);
}
不足：兼容性差，IE9及以上可用（若知道盒子的宽度用margin-left盒子宽度的一半的负值可解决兼容性问题）
水平居中

1.5 使用 flex布局 实现

.parent5-1 {
  display: flex;
  justify-content: center;
}
.parent5-2 {
  display: flex;
}
.child5-2 {
  margin: 0 auto;
}
不足：兼容性差，如果进行大面积布局可能会影响效率
水平居中

2 垂直居中

2.1 vertical-align

vertical-align 为 inline-block 依赖型元素（inline/inline-block才生效，table-cell可看作inline-block）
由于对齐的基线是用行高的基线作为标记，故需要设置 line-height 或设置 display: table-cell;
.parent {
  display: table-cell;
  vertical-align: middle;
  height: 20px;
}
.parent {
  display: inline-block;
  vertical-align: middle;
  line-height: 20px;
}
