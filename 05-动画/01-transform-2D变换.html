transform-2D变换

浏览器前缀

        -webkit-transition: color 1s; /* chrome & safari */
        -moz-transition: /* firefox */
        -o-transition: /* opera */

详细内容参考3D变换部分

        scale() [ scaleX() / scaleY() ]  缩放，基准为1，>1 放大 <1 缩小
            scale(n): 等比缩放
            scale(x,y): 水平/垂直缩放倍数
        translate() [ translateX() / translateY() ]  位移
            translate(x): 水平方向位移，垂直方向不变
            tarnslate(x,y): 水平/垂直方向上的位移
        rotate() [ rotateX() / rotateY() ]   旋转
            rotate(deg): 旋转角度，正值顺时针，负值逆时针
            [transform-origin]: 变换中心，默认几何中心，支持自定义：left right center top bottom ,40px 30px
        skew() [ skewX() / skewY() ]  倾斜
