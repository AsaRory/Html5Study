keyframe动画

定义1

    @keyframes <动画名>{
        0%{ }
        25%{ }
        ...
        100%{ }
    }

定义2

    @keyframes <动画名>{
        from{ }
        to{ }
    }

动画调用属性

    animation-name: @keyframes定义的动画名称
    animation-duration: 持续时间
    animation-timing-function: 运动曲线，具体见transtion过渡部分
    animation-delay: 0(默认), 延迟时间
    animation-iteration-count: n|infinite, 执行次数
    animation-direction: normal(默认)|alternate, 动画方向，alternate会在奇数次数(1,3,5...)正常播放，在偶数次数(2,4,6...)反向播放

    animation-play-state: paused|running, 可以设置初始暂停，在js中使用该属性可以在播放过程中暂停或播放动画
    animation-fill-mode : none|forwards|backwards|both, 动画播放前后动画效果是否可见，none不改变默认行为，forwards完成后保持最后一个属性值，backwards在delay阶段应用开始属性值，both向前和向后填充模式都被应用

缩写

    animation: name duration timing-function delay iteration-count direction;
