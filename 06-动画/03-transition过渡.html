transition

相关属性

    transition-property: background-color | all; // 作用的属性名
    transition-duration: 1s; // 过渡持续时间
    transition-timing-function: ease | linear | ease-in | ease-out | ease-in-out; // 过渡模式
    transition-delay: 1s; // 延迟执行时间

缩写

    transition: <过渡属性名称> <过渡时间> [<过渡模式> <延迟执行时间>]
    transition: all .1s ease-out 0s;

过渡模式 / 运动曲线 transition-timing-function

    ease    先加速后减速，缓慢开始，缓慢结束（默认）
    linear    线性匀速
    ease-in    加速，缓慢开始
    ease-out    减速，缓慢结束
    ease-in-out    缓慢开始，缓慢结束，和ease稍有区别
    cubic-bezier(n,n,n,n)   使用cubic-bezier函数中的值(0到1的数值)
