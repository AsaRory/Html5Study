CSS3.0 animation 3D场景

设置3D场景

    -webkit-perspective: 800; // 窗口距离三维物体800px
    -webkit-perspective-origin: 50% 50%; // 视点xy位置，此处表示正中央
    -webkit-transform-style: -webkit-preserve-3d; // 设置transform为3D

使用transform属性调整元素

    - translate (px)
      * translateX
      * translateY
      * translateZ
    - rotate (deg)
      * rotateX
      * rotateY
      * rotateZ

    - Y
