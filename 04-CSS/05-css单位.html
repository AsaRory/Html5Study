【 px 】：像素，绝对单位
【 em & rem 】
em：相对长度单位，参照当前元素字号大小，如果当前元素未设置字号则会继承其祖先元素字号大小
    例如：
    .box {font-size: 16px;} 则 1em = 16px
    .box {font-size: 32px;} 则 1em = 32px，0.5em = 16px

rem：相对长度单位，参照根元素(html)字号大小
    例如：
    html {font-size: 16px;} 则 1rem = 16px
    html {font-size: 32px;} 则 1rem = 32px，0.5rem = 16px;
    注：所有浏览器默认字号是16px，最小12培训，设置小于12px时，以12px计算
   【技巧】font-size: 62.5%; 因为 10/16 = 0.625

【 vh/vw & vmin/vmax 】
vh/vw：相对视窗的高度和宽度，1vh等于视窗高度的1%
vmin/vmax：vw和vh中较小或者较大的一个

【 rpx 】
rpx：小程序
