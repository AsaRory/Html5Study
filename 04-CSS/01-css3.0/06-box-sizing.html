box-sizing 盒模型： content-box border-box

box-sizing: content-box | border-box | inherit 

【 W3C标准盒模型 】 外加模式(CSS2.0默认值)，content-box，Element-width = width + border + padding

外盒尺寸计算（元素空间尺寸，包含margin）
element w/h ＝ content w/h ＋ padding ＋ border-width ＋ margin
200px + 30px * 2 + 20px * 2 + 30px * 2 = 360px

内盒尺寸计算（元素大小，不含margin）
element w/h ＝ content w/h ＋ padding ＋ border-width
200px + 30px * 2 + 20px * 2 = 300px

【 IE传统下盒模型（IE6以下，不包含IE6版本）】内减模式，border-box，Element-width = width

外盒尺寸计算（元素空间尺寸，包含margin）
element w/h ＝ content w/h ＋ margin
200px + 30px * 2 = 260px

内盒尺寸计算（元素大小，不含margin）
element w/h ＝ content w/h
200px

content-box: 默认值，W3C的标准盒模型，element width/height = border + padding + content width / height
border-box: IE传统的盒模型，元素的宽度高度等于元素内容的宽度高度，内容宽度或高度＝盒子的宽度或高度—边框—内距
inherit: 继承父元素的盒模型模式
box-sizing: content-box
box-sizing: border-box
